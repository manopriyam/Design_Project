## Setup Instructions : <br>

1. Clone the Repository
```
git clone https://github.com/manopriyam/Design_Project.git
```

2. BackEnd 
```
cd server
npm install
npm start
``` 

3. FrontEnd 
```
cd client
npm install
npm start
```
